# Book Specification

> **归档提示：** V6 文档，已废止；当前规范见 [`../../book/README.md`](../../book/README.md)。

## 目标

编写一本电子书级《现代 AI Agent 架构解析》，面向 AI Coding 开发者。

## 目标读者

-   初学者
-   AI 应用开发者
-   Agent Framework 开发者
-   IDE 插件开发者
-   企业平台开发者

## 写作原则

每个知识点必须说明 What / Why / How / When / Relationship。
