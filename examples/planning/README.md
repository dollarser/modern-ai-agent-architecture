# Planning - Agent 任务规划示例

## 学习目标
理解 Plan-and-Execute 模式、任务分解、步骤状态管理与执行流程

## 前置知识
- 第 5 章「Reasoning 与 Planning」
- 第 2 章「总体架构与生命周期」

## 运行方式

### Python
```bash
cd python
pip install -r requirements.txt
python main.py
```

### TypeScript
```bash
cd typescript
npm install
npm run start
```

## 预期输出
针对不同类型任务（搜索、计算、通用分析）自动生成执行计划，逐步执行并输出每步状态和结果。
