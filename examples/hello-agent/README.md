# Hello Agent - 最小 Agent 实现

## 学习目标
理解 Agent 的核心循环：Reasoning → Planning → Execute → Observe

## 前置知识
- 第 1 章「AI Agent 简介与历史演进」
- 第 2 章「总体架构与生命周期」

## 运行方式

### Python
```bash
cd python
pip install -r requirements.txt
python main.py
```

### TypeScript
```bash
cd typescript
npm install
npm run start
```

## 预期输出
Agent 执行一次完整的推理-行动-观察循环，输出每一步的状态。
